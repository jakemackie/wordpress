# Wordpress
A simple repository by [Jake](https://github.com/jakemackie) & [Alfie](https://github.com/alfiephillips) as a way of learning Wordpress.

## How to use
To clone and run this application, you'll need [Git](https://git-scm.com/download) and [Docker](https://www.docker.com/get-started/) installed on your computer. From your command line:
```bash
# Clone this repository
$ git clone https://github.com/jakemackie/wordpress

# Go into the repository
$ cd wordpress

# Start a container
$ docker compose up
```
Once the container's done building, visit https://localhost:8000 🎉
